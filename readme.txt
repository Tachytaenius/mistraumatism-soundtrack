This is for Linux. Could be ported to Windows or whatever, I'm sure.

Two copies of the ROM for LSDJ 9.4.0 are required. Name them:
lsdj9_4_0_Mistraumatism_OST_A.gb
lsdj9_4_0_Mistraumatism_OST_B.gb
(LSDJ ROMs may not be redistributed.)

You will also need the (release) binary for https://github.com/Tachytaenius/gb-presenter-rs/tree/clean-trim accessible with the command gb-presenter-rs-clean-trim.
And other things like wine. I also use the trash command from the trash-cli package. I don't much like using rm -rf...

You will very likely have to edit the scripts to point to the tools required locally to your machine.
This is only really here to show how I got it working.
Have a read through!
